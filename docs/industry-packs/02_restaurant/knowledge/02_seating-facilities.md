# 席・設備案内

| 項目 | 値 |
|------|-----|
| documentId | `seating-facilities` |
| タイトル | 席・設備案内 |
| 主な Layer | 1 / 2 |

## 役割

人数・シーン提案の条件。半個室・座敷・子ども椅子など「ある／ない／何室」を明確に。

## 入れるとよい内容

### 確定値

- テーブル・カウンター・座敷あり  
- 半個室2室（〜6名）。静かな席の候補だが確約は空席次第  
- 子ども椅子あり／全面禁煙  
- 空席の有無は書かない  

- 席種: テーブル、カウンター、座敷、半個室（室数・人数目安）  
- 車椅子対応、子ども椅子、ベビーカー置場  
- 「静かな席」の目安（半個室が候補、確約は空席次第）  
- 喫煙／禁煙  

## 推奨セクション

1. 席の種類と人数目安  
2. 個室・半個室  
3. 子ども・バリアフリー  
4. 利用上の注意

## 注意
