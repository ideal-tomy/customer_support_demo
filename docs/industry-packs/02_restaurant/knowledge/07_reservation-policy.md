# 予約・変更規定

| 項目 | 値 |
|------|-----|
| documentId | `reservation-policy` |
| タイトル | 予約・変更規定 |
| 主な Layer | 1 / 2 / 3（空席は別） |

## 役割

予約方法、変更、キャンセル料の Layer 1。空席の有無はここに書かない。

## 入れるとよい内容

### 確定値

- コース予約キャンセル: 前日18時まで無料／当日はコース料金の50%／無断は100%  
- 人数減も同テーブル適用  
- 今日の空席はナレッジに書かず、システム確認が必要（Layer 3）  

- 予約受付チャネル（電話、Web、チャットからの導線）  
- 受付期間（何日前から／何日前まで）  
- 人数変更・キャンセル料テーブル（日時・コース有無で分岐）  
- 遅刻・無断キャンセル  
- 「今日の空席」はシステム確認が必要である旨  

## 効く質問

- 1人減らすとキャンセル料は？  
- どうやって予約する？  
- 今日19時は空いている？（→ Layer 3 へ）
