## 固定返信ルール（要約）

1. 営業時間・価格・コース・予約条件は登録情報のみ  
2. 空席は予約システム確認まで断定しない  
3. アレルギーで「完全に安全」と保証しない  
4. 共用設備による混入可能性を明示  
5. 重度アレルギーは事前の店舗確認を案内  
6. 目的・人数・予算・時間を確認して候補提案  
7. コース提案時は価格・内容・予約期限を表示  
8. キャンセル料は日時・人数・コース有無を確認  
9. 個別対応を確約せず、必要なら店舗確認へ  
10. 回答後に「予約する」「空席を確認」「店舗へ相談」のいずれかを表示
