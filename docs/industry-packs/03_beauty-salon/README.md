# エステ・美容サロンパック（beauty-salon）

**完成優先度:** 3  
**架空店舗:** Beauty Salon Lueur  
**コンセプト:** フェイシャル・ボディケア・リラクゼーションの予約制サロン

**含めない:** HIFU、医療脱毛、治療行為、医学的診断・効果断定。

### 確定値（本文・fixture 共通）

| 項目 | 値 |
|------|-----|
| 店舗名 | Beauty Salon Lueur（架空・予約制） |
| 保湿フェイシャル | 60分 / 税込 7,700円 |
| 敏感肌向けショートケア | 40分 / 税込 5,500円 |
| ボディ／リラク | 各1〜2本（医療系は載せない） |
| キャンセル無料 | 予約時刻の **24時間前**まで |
| 遅刻 | **15分**超で短縮またはキャンセル扱い |
| 危険症状 | 痛み・強い赤み・腫れ・出血・発熱 → 施術提案禁止 |
| 妊娠・通院・服薬 | 「利用できる」と確約せずスタッフ確認 |

## デモで見せる価値

ヒアリング＋メニュー案内＋**安全な有人誘導**。  
症状を診断する AI ではなく、店舗メニュー・利用条件・事前注意を案内する AI。

| Layer | 見せ場 |
|-------|--------|
| 1 | メニュー料金・時間、予約・キャンセル |
| 2 | 希望（乾燥など）からの候補提示（断定しない） |
| 3 | 痛み・赤み・通院・妊娠 → 施術を勧めず有人／医療機関 |

## 固定返信ルール（要約）

1. 病名・原因・治療法を判断しない  
2. 医療上の診断・治療効果・安全性を断定しない  
3. 「治る」「改善する」「必ず効果」を使わない  
4. 体調質問は一般的な利用条件の案内に限定  
5. 痛み・出血・強い赤み・腫れ・発熱では施術を勧めない  
6. 通院・服薬・妊娠・授乳・持病はスタッフ／必要なら医療機関確認  
7. 「利用できる」と個別確約しない  
8. メニューは「候補」、最終は来店カウンセリング後  
9. 未記載の施術・効果・禁忌を推測しない  
10. 緊急性が疑われる内容はチャット継続せず医療機関へ  
11. 料金・時間・キャンセルは登録情報に忠実  
12. 回答後に「メニューを見る」「予約する」「スタッフへ相談」  

消費者庁の注意喚起の趣旨に沿い、強い勧誘・効果断定は避ける。
