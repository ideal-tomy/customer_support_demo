# 利用を控える条件

| documentId | `contraindications` |
| タイトル | 利用を控える条件 |
| Layer | 1 / 3 |

## 役割
発熱、炎症、妊娠、通院中など「控える／要確認」のリスト。個別に「受けられる」と書かない。

## 入れる内容

### 確定値

- 痛み・強い赤み・腫れ・出血・発熱がある場合は **施術を勧めない**（メニュー提案禁止）  
- 妊娠・授乳・通院・服薬は「利用できる」と確約せず、スタッフ確認を案内  

- 発熱・感染症の疑い  
- 施術部位の強い炎症・傷  
- 妊娠中・授乳中（メニューにより要確認）  
- 皮膚科等通院中・服薬中はスタッフ確認  
- チャットでは最終可否を断定しない
