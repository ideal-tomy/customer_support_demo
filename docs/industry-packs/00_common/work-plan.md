## やらないこと（この計画のスコープ外）

- 業種ごとに別リポジトリ・別アプリを増やすこと
- 本番在庫・予約・配送 API の本接続（デモはモックで十分）
- 配送業者向けデモの先行実装（小売の注文・配送部分からの横展開は後続）
