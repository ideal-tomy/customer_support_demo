```text

```

## 最も魅力的な連続体験（業種別）

### 小売

1. 返品期限は何日ですか？ → 単一 FAQ（Layer 1）  
2. 開封したが初期不良。返品できるか？ → 返品×保証横断（Layer 2）  
3. 注文した商品の現在地 → 配送データ（Layer 3）  

### 飲食

営業時間（L1）→ 条件からコース提案（L2）→ 空席確認して予約（L3）

### エステ

メニュー回答（L1）→ 希望から候補提案（L2）→ 危険・個別質問は有人（L3）
