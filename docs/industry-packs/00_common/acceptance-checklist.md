# 受け入れチェックリスト（業種パック）

手動確認用。自動テストランナーはない。業種選択 → チャット →「最初からやり直す」（同一業種で会話リセット）／アバター（業種選択へ）の導線を先に確認する。

---

## 共通

- [x] 起動直後は業種カード3つ（チャットはまだ出ない）
- [x] カード選択で該当ブランドのチャットが始まる
- [x] ヘッダーに「最初からやり直す」があり、押すと選択中業種のまま会話が初期化される
- [x] ヘッダー左のアバターを押すと業種選択に戻る
- [x] サンプル会話は回答後も用件チップ／関連質問で最大3往復まで続けられる
- [x] Knowledge シートに業種タブはない（店舗のご案内／お試しナレッジのみ）
- [x] 通常モード: 回答下に「参照したご案内: …」（citations がある場合）
- [x] デモ説明モード OFF: 技術向け「回答方式」は出ない
- [x] 設定でデモ説明モード ON: 「回答方式: …」が出る（session 内保持）
